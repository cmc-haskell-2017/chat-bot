Чат-бот

Чат-бот - серверная программа, связь с которой происходит по локальной сети с использованием расширенного протокола SMTP.
Тут же имеется клиент для общения с ботом. Клиент автоматически присоединяется к серверу. 

Как пользоваться клиентом:
Все, что написано в stdin будет приконкатенированно к буферу. В начальный момент буфер пуст.
Команды для управления буфером
 - /send - отправлетя текущее содержимое чат-боту
 - /reset - очищает буфер
 - /end - завершает использование программы

Что умеет делать чат-бот?
Получает одной строкой целые числа, после чего отправляет в ответ их сумму.
